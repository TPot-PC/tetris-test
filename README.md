# Game Boy Style Tetris

懐かしいゲームボーイ風のデザインと操作感を再現したWeb版テトリスです。

![Game Screen](./screenshot.png)

## 特徴

*   **レトロなデザイン**: クラシックな緑色の4階調パレット。
*   **忠実な操作感**:
    *   PC: キーボード（矢印キー, Z/Xキー）
    *   スマホ: 仮想D-Padとボタン
*   **オンラインランキング**: Google Sheetsをバックエンドに使用したハイスコア機能。
*   **パスワード保護**: 特定のユーザーのみアクセス可能なロック機能（SHA-256）。

## 操作方法

| アクション | キーボード | 画面ボタン |
| :--- | :--- | :--- |
| **移動** | ← / → | D-Pad 左右 |
| **ソフトドロップ** | ↓ | D-Pad 下 |
| **回転 (右)** | X / ↑ | Aボタン / D-Pad 上 |
| **回転 (左)** | Z | Bボタン |
| **スタート/ポーズ** | Enter | Start |
| **ミュート** | M | - |
